## Scene Checklist

### Every scene should:
- [ ] Have a clear goal
- [ ] Contain conflict
- [ ] End with change
- [ ] Move the plot OR reveal character
- [ ] Include at least 2 sensory details
- [ ] Contain at least one moment of tension

### Should NOT:
- [ ] Be only description
- [ ] Repeat information we already know
- [ ] Leave the character unchanged
- [ ] Be disconnected from the plot

## Scene Writing Techniques

### Scene Opening
- Enter as late as possible
- Start with action or dialogue
- Signal the conflict immediately

### Scene Middle
- Escalate the conflict
- Attempts and failures
- Unexpected complications

### Scene Ending
- Exit as early as possible
- Leave on a peak of tension
- Don‚Äôt over-explain consequences

### Scene Transitions
- Time jump: ‚ÄúThree days later‚Ä¶‚Äù
- Location jump: new chapter = new place
- POV switch: change the character
